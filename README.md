# PathWander - Property Management Web Application

PathWander is a dynamic web application that allows users to seamlessly sign in, create, edit, and update property listings as well as allows user to give a review/rating in each listing. This project emphasizes a user-friendly interface and is built using Node.js, Express, and MongoDB. Frontend features include CSS styling + Bootstrap Styling, JavaScript for interactivity, and EJS templating for dynamic content rendering.

## Key Features
- **User Authentication:** Secure sign-in functionality using Passport.js for authentication.
- **Property Management:** Create, edit, and update property listings with ease.
- **RESTful Architecture:** Follows RESTful principles for efficient communication between frontend and backend.
- **Data Security:** Ensures data integrity and security in MongoDB, protecting sensitive information.
- **Middleware Integration:** Utilizes middleware for smooth session management, flash messages, and error handling.

## Hosting
The application is hosted on Render, providing accessibility and scalability for a reliable user experience.

## Technologies Used
- **Backend:** Node.js, Express, MongoDB
- **Frontend:** EJS Templating, CSS, Bootstrap for styling, JavaScript.
- **Authentication:** Passport.js
- **Hosting:** Render

## Getting Started
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Set up your MongoDB database.
4. Configure environment variables, including MongoDB connection details and session secret.
5. Run the application using `nodemon app.js`.
6. Access the application at `http://localhost:8080`.
